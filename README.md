# 🌦️ Live Weather Dashboard – India (Power BI)

## 🔹 Project Overview
This Power BI dashboard provides real-time weather insights across major Indian cities. 
It visualizes temperature, humidity, wind speed, and weather conditions using interactive reports.

## 🔹 Key Features
- 🌍 City-wise weather map visualization
- 🌡️ Temperature monitoring
- 💧 Humidity tracking
- 🌬️ Wind speed analysis
- 📊 KPI cards for live weather metrics
- 🔎 City-level filtering

## 🔹 Tools Used
- Power BI
- Power Query
- DAX
- Excel / CSV Dataset

## 🔹 Skills Demonstrated
Data Cleaning | Data Modeling | DAX Measures | Interactive Dashboard Design | Data Visualization

## 🔹 Data Source
Live weather data fetched dynamically using OpenWeatherMap API.
The API key is securely stored and not exposed in this repository.

## 🔹 Technical Implementation
- Used Web connector in Power BI to fetch live JSON data
- Transformed JSON using Power Query
- Created DAX measures for KPI visualization
